# Meme Token Hunter

Twitter-based hot meme coin detector with real-time monitoring.

## Features
- Multi-timeframe monitoring (1h, 24h, 7d)
- Twitter trend analysis
- Contract address verification
- KOL activity tracking
- Risk assessment (LOW/MEDIUM/HIGH/CRITICAL)
- Anti-duplicate tracking

## Usage
```bash
npm install
node hunter.js
```

## Configuration
Edit `config.json` to customize:
- Search keywords
- KOL accounts to track
- Alert thresholds
- Monitored blockchains (SOL/ETH/BASE/BSC)
