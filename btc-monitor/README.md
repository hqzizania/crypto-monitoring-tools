# BTC Market Monitor

Real-time Bitcoin price monitoring with technical analysis and AI-powered market insights.

## Features
- Multi-timeframe K-line analysis (5m, 15m, 1h, 4h)
- RSI and MA indicators  
- Volume spike detection
- Twitter sentiment analysis
- Macro economic news integration
- Automated hourly reports

## Usage
```bash
npm install
node monitor.js    # Basic price + technical analysis
node analyzer.js   # Full AI analysis with web search
```

## Configuration
Edit `config.json` to customize timeframes, alert thresholds, etc.
